# Lab 4: Disk Space & Storage Troubleshooting

[Recording Link](https://youtu.be/cyKSUUof4-U)

## Objective
Identify, diagnose, and resolve a low disk space issue affecting a standard user in a Windows environment.

## Environment
- Windows 11 (VMware)
- Local accounts:
  - ITADMIN (Administrator)
  - user1 (Standard User)

## Actions Performed
- Checked available disk space on the system drive.
- Created a controlled low-disk-space condition.
- Logged in as a standard user to observe user impact.
- Diagnosed disk usage as an administrator.
- Used Disk Cleanup and terminal to remove unnecessary files.
- Verified restored disk space and normal user functionality.

## Outcome
Low disk space condition was resolved successfully. Disk space warnings were cleared, and the standard user was able to operate normally.

## Skills Demonstrated
- Disk usage monitoring
- Disk Cleanup and storage management
- User-impact troubleshooting
- Safe file removal practices
- Windows Help Desk fundamentals
