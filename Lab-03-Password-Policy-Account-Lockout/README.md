# Lab 3: Password Policy & Account Lockout

[Recording Link](https://youtu.be/tKLCdhbqvfM)

## Objective
Configure and test local password policies and account lockout behavior in Windows.

## Environment
- Windows 11 (VMware)
- Local users: ITADMIN (Admin), user1 (Standard)

## Actions Performed
- Configured password complexity and length policies
- Set account lockout thresholds
- Tested weak password rejection
- Triggered account lockout via failed logins
- Unlocked user account through Local Users and Groups

## Outcome
Password and lockout policies were enforced correctly, and the locked account was successfully restored by an administrator.

## Skills Demonstrated
- Local Security Policy management
- Password policy enforcement
- Account lockout troubleshooting
- User account recovery
