# Lab 2: Local Users & Groups

[Recording Link](https://youtu.be/c8-mvTENJv8)

## Objective
Manage folder access using local security groups instead of individual user permissions.

## Environment
- Windows 11 (VMware)
- Local accounts:
  - ITADMIN (Administrator)
  - user1 (Standard User)

## Actions Performed
- Created a custom local security group.
- Added a standard user to the group.
- Created a test folder for group-based access.
- Assigned Modify NTFS permissions to the group.
- Verified access by logging in as the standard user.
- Confirmed file creation, modification, and deletion through group membership.

## Outcome
Access was successfully controlled through group membership, allowing permissions to be managed without modifying individual user accounts.

## Skills Demonstrated
- Local user and group management
- Group-based NTFS permissions
- Role-based access control
- Access verification and testing
- Windows access provisioning practices
